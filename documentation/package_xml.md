# Package.xml ([link](http://wiki.ros.org/catkin/package.xml))

The package manifest is an XML file called package.xml that must be included with any catkin-compliant package's root folder.
* The file define the properties about the package:
  * package name
  * version numbers
  * authors, maintainers 
  * and dependencies on other catkin packages
